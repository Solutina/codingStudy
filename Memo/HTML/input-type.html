input 타입 별 형태

타입 결과
text
password
number
email @가 포함되지 않을 경우 오류를 안내한다.
search 'X' 표시, 줄바꿈문자 제거 등 text와 일부 차이가 있다.
tel 검증은 하지 않는다. 그래서 pattern 속성과 함께 사용한다.
url "http://"가 들어가는지 검증해준다.
file
radio
name으로 묶여진 값 중에 하나만 선택이 가능하다.
checkbox
name을 통해 여러 선택을 함께 받을 수 있다.
date "년-월-일" 형식으로 데이터 전송
month "년-월" 형식으로 데이터 전송
week "년-W주차" 형식으로 데이터 전송
time "시:분" 형식으로 데이터 전송
datetime-local "yyyy-MM-ddThh:mm" 의 포맷으로 데이터 전송 (시간대 없음)
color 000000~ffffff까지의 값 입력
range
min=0, max=10, value=5일 때
button
submit '제출'을 눌렀을 때 form의 method에 맞는 방식으로 전송 (기본 : GET)
image text 형식이 아닌 image 형식의 submit.
hidden
hidden을 통해 전송한 필드는, 사용자에게 노출되지 않는다.
주로 토큰 값이나, 데이터베이스의 레코드를 저장한다.
